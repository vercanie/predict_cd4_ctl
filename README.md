# Analysis of scRNAseq data from T cells in the PREDICT study
![Summary of the paper](./pics/predict_workflow.jpg) 

This repository contains an analysis of the single-cell RNA sequencing of gut biopsies and PBMC samples from children with Crohn's disease (CD), ulcerative colitis (UC), 
disorders of gut-brain interaction (DGBI) and healthy donors within the PREDICT study. The manuscript is in development, please see our previous publications:

* [Concerted changes in the pediatric single-cell intestinal ecosystem before and after anti-TNF blockade, eLife, 2023](https://elifesciences.org/reviewed-preprints/91792#tab-content)
* [Divergent T Cell Phenotypes Define Pediatric Crohnâ€™s Disease and Ulcerative Colitis, medRxiv, 2025](https://www.medrxiv.org/content/10.1101/2025.10.16.25336112v1)

## Requirements
Running the code requires R version 4.2.1 and higher. Specific package versions might be necessary. For more instructions, please take a look at the SessionInfo included in each analysis file. 

## Instructions
For running the code, please clone the repository and download the preprocessed data. Data will be made available at Gene Expression Omnibus upon publication of the manuscript.
